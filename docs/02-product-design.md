## 一、产品架构总览

```
BoxingCat（拳击猫）
│
├── 🏠 桌面层（Desktop Layer）

│   └── 通知（飞盘、猫屎、旅行归来）
│
├── 🥊 健身系统（Fitness System）
│   ├── 摄像头动作捕捉
│   ├── AI 拳击姿势评分
│   ├── 训练记录与进度
│   └── 猫粮产出
│
├── ✈️ 旅行系统（Travel System）
│   ├── 猫猫自动出门
│   ├── 随机事件引擎
│   └── 归来带回物品/消息
│
├── 🥏 飞盘社交（Frisbee Social）
│   ├── 飞盘购买/发射
│   ├── 飞盘飞行路由
│   ├── 捕获判定（敏捷度）
│   ├── 猫屎惩罚
│   └── 经济循环

├── 🏛️ 广场系统（Plaza — 远期）
│   ├── 任务板
│   ├── 兑换商店
│   ├── 社区入口
│   └── DAO 治理
│
└── 🔐 AAStar 账户体系（Account Abstraction）
    ├── 主人账户（Master Account）— 邮箱/社交登录，Gasless
    ├── 猫猫 Agent 账户（派生自主人账户）
    └── NFT 资产（外观/道具）— AA 智能合约钱包管理
```

## 二、用户生命周期

### 2.1 新用户上手指南

```
第 0 步：下载安装 BoxingCat 桌面应用
第 1 步：邮箱/社交账号注册（AAStar 自动创建 AA 智能合约钱包）
第 2 步：购买初始 NFT（猫的基础外观）— Gasless 交易
第 3 步：自动生成猫猫 Agent 账户（从主人账户派生）
第 4 步：猫猫出现在桌面，进入新手引导
第 5 步：完成第一次拳击训练（送初始猫粮）
第 6 步：解锁日常循环
```

### 2.2 日常循环

```
每天打开电脑
  → 猫猫在桌面活动
  → 做拳击训练（5-15 分钟）
    → 获得猫粮 + 敏捷度提升
  → 消耗猫粮买飞盘 / 猫粮消费
  → 猫猫偶尔出门旅行
  → 可能收到别人飞来的飞盘
  → 第二天继续
```

### 2.3 长期目标

| 短期（每日） | 中期（每周） | 长期（月度+） |
|-------------|-------------|--------------|
| 完成拳击训练 | 猫粮积累到一定量 | NFT 升级/换新外观 |
| 获得猫粮 | 成功捕获飞盘 | 敏捷度排名 |
| 喂猫/互动 | 猫猫旅行归来 | 广场参与治理 |

## 三、经济系统设计

### 3.1 代币体系

```
猫粮（Cat Food / $CF）
  — 应用内主要经济代币
  — 通过锻炼获得（核心产出）
  — 通过捕获飞盘获得
  — 消耗于：购买飞盘、打扫猫屎、购买道具、NFT 相关

敏捷度（Agility Points / AP）
  — 不可交易属性
  — 仅通过拳击训练提升
  — 影响飞盘捕获成功率
  — 可能影响旅行事件结果
```

### 3.2 猫粮经济循环

```
                    ┌──────────────┐
                    │   拳击训练    │
                    │  （核心产出）  │
                    └──────┬───────┘
                           │ +猫粮, +敏捷度
                           ▼
    ┌──────────────────────────────────────────┐
    │              用户猫粮余额                  │
    └───┬────────────┬────────────┬────────────┘
        │ 消耗       │ 消耗        │ 消耗
        ▼            ▼             ▼
    ┌────────┐  ┌──────────┐  ┌──────────┐
    │ 买飞盘  │  │ 打扫猫屎  │  │ NFT/道具  │
    └───┬────┘  └──────────┘  └──────────┘
        │
        ▼
    ┌────────────────┐
    │ 飞盘飞到别人家  │
    │                │
    │ 对方捕获成功    │ → 对方 +猫粮
    │ 对方捕获失败    │ → 对方 -猫粮（打扫）
    └────────────────┘
```

### 3.3 飞盘经济学

| 场景 | 发送方 | 接收方 |
|------|--------|--------|
| 购买飞盘 | -N 猫粮 | — |
| 捕获成功 | — | +M 猫粮（M > N 则有净收益） |
| 捕获失败 | — | -X 猫粮（打扫猫屎费用） |
| 飞盘没被任何人捕获 | 可能退回部分猫粮 | — |

**关键设计决策**：飞盘的捕获收益应大于购买成本（激励锻炼），但打扫成本不应太高（避免挫败感）。

## 四、各系统详细设计

### 4.1 拳击健身系统

#### 训练流程

```
1. 用户点击"开始训练"
2. 选择训练模式：
   - 自由模式（不限动作，跟着节奏出拳）
   - 课程模式（跟随教练动作）
   - 挑战模式（限时高难度组合拳）
3. 摄像头打开，AI 模型加载
4. 实时动作捕捉 → 骨架关键点识别
5. 每组动作完成后即时评分
6. 训练结束 → 汇总统计 + 猫粮/敏捷度结算
```

#### 评分维度

| 维度 | 说明 | 权重 |
|------|------|------|
| 姿势准确性 | 出拳角度、身体旋转 | 40% |
| 力量感 | 速度 + 加速度曲线 | 25% |
| 连贯性 | 组合拳的流畅度 | 20% |
| 耐力 | 训练总时长/总次数 | 15% |

#### 拳击动作库（MVP）

```
基础动作：
  - 直拳（Jab / Cross）
  - 勾拳（Hook）
  - 上勾拳（Uppercut）
  - 闪避（Slip / Roll）

进阶动作：
  - 组合拳（1-2 combo, 1-2-3 combo）
  - 步法移动
```

### 4.2 猫猫桌面表现

#### 状态机

```
┌─────────┐    锻炼后     ┌─────────┐
│   idle   │──────────────▶│  active  │
│ （桌面待机）│              │ （活跃中） │
└────┬─────┘              └────┬─────┘
     │                         │
     │ 随机/定时               │ 一段时间后
     ▼                         ▼
┌─────────┐              ┌─────────┐
│  travel  │              │  sleep   │
│ （旅行中）│              │ （睡觉）  │
└────┬─────┘              └─────────┘
     │ 归来
     ▼
┌─────────┐
│ return  │
│（带回消息）│
└─────────┘
```

#### 桌面交互

| 交互 | 猫猫反应 |
|------|---------|
| 点击猫猫 | 被戳反应动画 + 随机对话气泡 |
| 双击 | 打开主面板 |
| 拖拽 | 猫猫被拖到屏幕任意位置 |
| 右键菜单 | 设置/训练/查看状态 |
| 长时间不动 | 猫猫自己走来走去/做随机动作 |
| 飞盘飞来 | 猫猫跳跃尝试捕获（基于敏捷度） |
| 猫屎掉落 | 猫猫嫌弃的表情/动作 |

#### 外观系统

```
NFT 决定：
  - 猫的基础品种（英短/美短/布偶/橘猫...）
  - 毛色/花纹
  - 眼睛颜色
  - 配饰（帽子/眼镜/项圈...）

可装备（可能也是 NFT）：
  - 拳击手套样式
  - 服装
  - 特效粒子
```

### 4.3 旅行系统

#### 参考设计

参照《旅行的青蛙》的核心体验：

| 旅行的青蛙 | BoxingCat |
|-----------|-----------|
| 青蛙自己出门 | 猫猫自己出门 |
| 带便当决定时长 | 消耗猫粮决定旅行时长/目的地 |
| 寄明信片回来 | 带消息回来（推送通知） |
| 带回特产 | 带回随机物品/猫粮/NFT 碎片 |
| 可能遇到朋友 | 可能遇到其他用户的猫 |

#### 旅行配置

```
用户可配置：
  - 旅行时长（短途/中途/长途）
  - 消耗猫粮量（更多猫粮 = 更远/更好的随机结果）
  - 装备旅行道具（如果有）

猫咪自主：
  - 什么时候出门（随机/用户触发）
  - 目的地
  - 遇到的事件
  - 带回来的东西
```

#### 旅行事件表（示例）

| 事件类型 | 描述 | 可能收获 |
|---------|------|---------|
| 公园散步 | 猫在公园遇到了其他猫 | 低量猫粮、社交记录 |
| 迷路 | 猫走丢了，需要时间找回来 | 延迟归来 |
| 发现宝藏 | 在垃圾桶翻到好东西 | 稀有道具碎片 |
| 打架 | 和其他猫发生冲突 | 可能受伤（需要猫粮治疗） |
| 拍照 | 在风景好的地方自拍 | 带回照片（可分享） |
| 遇到飞盘 | 在旅途中拦截了一个飞盘 | 额外猫粮 |

### 4.4 飞盘社交系统

#### 飞盘生命周期

```
┌─────────┐
│ 用户购买  │  消耗猫粮
└────┬────┘
     │
     ▼
┌─────────┐
│ 飞盘发射  │  自动随机路由到其他在线用户
└────┬────┘
     │
     ▼
┌──────────────┐
│ 飞盘到达目标  │  对方桌面出现飞盘动画
└────┬─────────┘
     │
     ├── 敏捷度 ≥ 阈值（当前飞盘稀有度决定）
     │   ┌──────────┐
     │   │  捕获成功  │  → 对方获得猫粮/飞盘资产
     │   └──────────┘
     │
     └── 敏捷度 < 阈值
         ┌──────────┐
         │  捕获失败  │  → 飞走 + 丢猫屎
         └──────────┘
              │
              ▼
         ┌──────────┐
         │ 猫屎清理  │  消耗猫粮打扫
         └──────────┘
```

#### 飞盘稀有度

| 稀有度 | 所需敏捷度 | 捕获奖励 | 猫屎打扫费 |
|--------|-----------|---------|-----------|
| 普通 | 低 | 少量猫粮 | 低 |
| 稀有 | 中 | 中量猫粮 + 可能道具 | 中 |
| 传奇 | 高 | 大量猫粮 + 稀有道具 | 高 |
| 黄金 | 极高 | 巨额猫粮 + NFT 碎片 | 很高 |

#### 防刷机制

- 飞盘每日购买/发射有上限
- 同一个用户不能频繁向同一目标发送
- 敏捷度需要真实锻炼数据支撑（不可直接购买）

## 五、账户体系选型

### 5.1 不使用 Solana 的理由

BoxingCat 的目标用户是宅男程序员/二次元群体，**大多数不是加密原生用户**。直接暴露链上操作（助记词、Gas 费、RPC 节点）会严重阻碍转化。

### 5.2 AAStar SDK 方案

```
npm:     @aastar/sdk
仓库:    AAStarCommunity/aastar-sdk
定位:    Account Abstraction (ERC-4337) + Gasless + Web3

核心能力：
  ✅ 邮箱/社交登录 → 自动创建 AA 智能合约钱包
  ✅ Gasless 交易（用户无需持有任何代币即可操作）
  ✅ 内置社交恢复（丢手机也能找回）
  ✅ TypeScript SDK，与 Tauri Web 技术栈一致
  ✅ 猫粮代币、NFT 资产均在 AA 账户内管理
```

### 5.3 账户架构

```
用户注册流程：
  邮箱登录 → AAStar SDK 本地生成 AA 钱包
    → 部署智能合约账户（Gasless，AAStar 代付）
    → 账户即主人账户

猫猫 Agent 账户：
  → 从主人 AA 账户派生（无需额外注册）
  → 不持有私钥，由主人账户控制
  → 存储链上属性：敏捷度、等级、NFT 持有

NFT 购买：
  → 用户点击购买
  → AAStar SDK 发起 Gasless 交易
  → NFT 铸造到主人 AA 账户
  → 猫猫外观自动切换
```

## 六、用户画像

### 主要用户：宅男程序员 / 二次元群体

| 特征 | 描述 |
|------|------|
| 年龄 | 18-35 岁 |
| 职业 | 程序员/设计师/学生 |
| 生活习惯 | 长时间面对电脑、缺乏运动 |
| 兴趣 | 二次元、养宠、游戏、NFT |
| 痛点 | 想锻炼但缺乏动力和趣味性 |
| 动机 | 喜欢养成感 + 不希望锻炼太"正经" |
| 消费意愿 | 愿意为 NFT/皮肤/道具付费 |

### 用户场景

**张三（26岁，后端程序员）**
- 每天早上 9 点到公司打开电脑，猫猫出现在桌面右下角
- 10 点半做一组 5 分钟拳击训练，拿到 50 猫粮
- 用猫粮买了普通飞盘，随便发射了一下
- 下午发现猫猫出门旅行了，带回来一张"海边夕阳"照片
- 晚上收到通知：飞盘被另一个用户捕获了，对方回赠了 10 猫粮作为感谢
- 积累一周的敏捷度后，捕获了一个稀有飞盘，获得限量拳击手套皮肤
